# UpSkill Campus - Digital Marketplace Platform

A full-stack web application for a digital marketplace where merchants can offer services and customers can browse, purchase, and leave reviews. Built with the MERN stack with Razorpay payment integration.

## 🌟 Features

### Customer Features
- **User Authentication**: Secure signup and login with JWT
- **Service Browsing**: Browse and search services offered by merchants
- **Shopping Cart**: Add/remove services from cart before checkout
- **Payments**: Secure payment processing with Razorpay integration
- **Order Management**: Track and view order history and details
- **Reviews & Ratings**: Leave reviews and ratings for purchased services
- **Client Portal**: Personalized dashboard to manage purchases and account

### Merchant Features
- **Service Management**: Create, update, and manage service offerings
- **Order Tracking**: View and manage incoming orders
- **Revenue Dashboard**: Monitor sales and earnings
- **Merchant Dashboard**: Analytics and performance metrics

### Admin Features
- **User Management**: Block/unblock users and merchants
- **Platform Analytics**: View overall platform statistics
- **Order Management**: Monitor all orders on the platform
- **Content Moderation**: Review and manage user-generated content

## 🛠️ Tech Stack

### Backend
- **Node.js & Express.js** - Server framework
- **MongoDB & Mongoose** - Database
- **JWT** - Authentication
- **Razorpay** - Payment gateway
- **bcryptjs** - Password hashing
- **Nodemailer** - Email notifications
- **CORS** - Cross-origin resource sharing

### Frontend
- **React 19** - UI framework
- **React Router v7** - Client-side routing
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **Axios** - HTTP client
- **React Icons** - Icon library
- **jsPDF** - PDF generation
